# Projeto_impacta
Criação de Software 
🔹 Arquitetura em 3 Camadas

Front-end (Interface do Usuário)

Tela de login (opcional).

Tela de cadastro de clientes (nome, CPF/CNPJ, telefone, e-mail, endereço).

Tela de listagem com busca e filtros.

Tela de edição/atualização.

Frameworks possíveis: React, Vue.js ou até HTML/CSS/JS + Bootstrap.

Back-end (API / Regra de Negócio)

CRUD (Create, Read, Update, Delete) para clientes.

Validações (ex.: CPF válido, e-mail no formato correto).

Autenticação de usuários (opcional, se quiser avançar mais).

Tecnologias: Node.js + Express, Django, ou Spring Boot.

Banco de Dados (Persistência)

Tabela Clientes com os campos:

id (PK)

nome

cpf_cnpj

email

telefone

endereco

data_cadastro

Banco recomendado: MySQL, PostgreSQL ou SQLite (se for mais simples).
